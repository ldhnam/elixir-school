About Elixir

“Elixir - динамический, функциональный язык программирования, разработанный для создания масштабируемых и легко поддерживаемых систем.” — elixir-lang.org

Elixir основан на проверенной временем виртуальной машине Erlang VM (BEAM) для разработки распределенных устойчивых к ошибкам систем с низким временем ответа.

Особенности:

  • Масштабируем
  • Устойчив к ошибкам
  • Функциональное программирование
  • Расширяем
