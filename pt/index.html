Sobre o Elixir

“Elixir é uma linguagem dinâmica e funcional projetada para construir aplicações escaláveis e de fácil manutenção.” — elixir-lang.org

Elixir aproveita a massivamente testada máquina virtual do Erlang para construir sistemas distribuídos e tolerantes a falhas com baixa latência.

Características:

  • Escalável
  • Tolerante a falhas
  • Programação funcional
  • Extensível
